The Librarian — Web Clipper

Save what you're reading straight into your Librarian.

The Librarian is a self-hosted, open-source memory layer for AI coding agents: durable, cross-session memory and reference material your agents can search, that travels with you across tools. Learn more at https://codeministry.net/the-librarian/

This extension is its web clipper. On any article, click the toolbar button — it extracts the main content (stripping nav, ads, and clutter), converts it to clean Markdown, and files it into your vault as a reference your agents can find later. No copy-paste, no leaving the page.

SETUP (about a minute)

1. Open your Librarian dashboard → Settings → Connect a device.
2. Copy your Server URL, and mint a capture token there.
3. Open this extension's Options, paste both, and save.

Then click the clipper on any page you want to keep.

PRIVACY

• Page content is sent only to YOUR Librarian server — the one you configure. Never to us, never to any third party. Nothing is sold or shared.
• It uses a capture-scoped token that can only file references; it cannot read your memories or reach any agent tools.
• Your server URL and token are stored locally in your browser.

REQUIREMENTS

A running Librarian server (self-hosted) and a capture token from its dashboard. Works in Chrome and Edge.
